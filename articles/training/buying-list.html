Buying List

Triathlon is such an expensive sport, so sorry to keep adding.

Swimming

Pull Buoy - bigger the better. I like the HUUB big buoy. Particularly for weaker swimmers (ie me) which allow us to get fit and strong without have to over think body position. Link here

Band / Ankle Lock - I use the Finis brand one and put it on during pulling sets, I use a figure of 8 when putting on. Too loose otherwise which allows kicking. Defeats the purpose. Link here

Paddles - Use paddles to not only develop strength but also at times limit how much we can aerobically whack ourselves. Most paddles way too big and significantly slow down cadence. I like the TYR Catalyst Green. Finis agility is also very good.

Swim Fins - help develop a more effective kick. Use during warm up s+ kick sets to help swim mechanics.

Tempo Trainer - I need to work on my cadence so use the Finis Tempo Trainer to help me dial that in. It’s also very good for the red mist type sessions we do as we near our goal race. Link here

Body Maintenance

Massage balls - Find these great. Really good for working Piriformis, Gastroc, Soleus. You get 3 in a pack, so can leave one in work etc. Link here

Lacrosse Ball - This is awesome for back. Link here

Larger Massage Ball - You need a bigger ball for Psoas (you need to be working this!). Link here

Foam Roller - Quads. I use either Trigger Point Therapy one, or this one Other

Headphones - Love these Plantronics. Don’t leave my house for a run without them. Fit nicely in my ear. No messing about fiddling during run. Link here

Blender - Must for post exercise. I have the this Nutribullet. Really recommend. Does a fine job on crushed avocado’s and other frozen fruit. Think it’s a model up from previous versions. Motor seems stronger.

Salt tab holder - Found these great in Hamburg + Cascais last year. Link here
